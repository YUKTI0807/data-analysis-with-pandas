# 🥑 Avocado Data Analysis using Pandas

## 📌 Project Overview
This project performs data cleaning, exploration and visualization on the Avocado dataset using **Python, Pandas and Matplotlib**.

The goal of this practical is to understand how to handle real-world datasets and extract meaningful insights using Pandas.

---

## 📂 Dataset
Avocado dataset contains information about:
- Average price
- Total volume
- Different bag sizes
- PLU codes (4046, 4225, 4770)
- Region
- Type of avocado

---

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Google Colab / Jupyter Notebook

---

## 🔍 Steps Performed

### ✅ Data Understanding
- Displayed first few rows using `head()`
- Checked dataset shape
- Viewed column names
- Generated statistical summary using `describe()`

### 🧹 Data Cleaning
- Checked missing values
- Removed null values
- Filled missing values
- Renamed columns for better readability

### 📊 Data Analysis
- Calculated:
  - Mean
  - Sum
  - Count
  - Minimum & Maximum values
- Used `value_counts()` for categorical analysis
- Found unique values
- Applied filtering with conditions
- Performed correlation analysis

### 📈 Data Visualization
- Scatter plot → Average Price vs Total Bags
- Histogram → Distribution of PLU 4046
- Line plot → Groupby analysis

### 🔄 Feature Engineering
- Created a new column:
  
  AveragePrice × Total Volume

---

## 📌 Key Insights
- Relationship between price and bag sales
- Region-wise average sales
- Most frequent avocado type
- Distribution of product volume
